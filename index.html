HTML

  • The head element contains information about the web page
  • The body element represents the visible content shown to the user

CSS

  • Margin indicates how much space we want around the outside of an element.
  • Padding indicates how much space we want around the content inside an element.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: Adds all changes in the repo
  • git add .: Adds all changes in the current and sub directories
  • git checkout main: switch to main branch
  • git commit -m "insert message": to commit changes with a message
  • git pull origin main: to make sure local environment is up to date
  • Stage changes, commit changes, make sure local is up-to-date, push to repo, pull request, merge changes

Javascript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • Strings are enclosed in quotation marks
  • The script element goes just before the closing body element
  • if statements are a way of introducing decision-making into code and will only execute if the statement is true
  • An array is a single variable that is used to hold a group of data and is typically used to hold data that is related in some way
  • Indexes start at zero
  • For loops perform a task on all the items in an array by allowing a single code block to be executed over and over.
